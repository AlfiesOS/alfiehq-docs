## Creating your first product

<Info>
  You must have the **admin** role to create products
</Info>

<Steps>
  <Step title="Run the command">
    ```
    /add-product <product_name> <role>
    ```

    \<product_name\> = The name of your product (eg. V2 Door System)

    \<role\> = The role ID to be appied to users when they are whitelisted this product (To see a product channel, chat with other buyers etc)

    Full Example

    ```
    /add-product <v2 door system> <1380678177327943730>
    ```
  </Step>
  <Step title="All Done!" icon="check">
    You have made your first product\!
  </Step>
  <Step title="Common Errors" icon="triangle" iconType="solid">
    
  </Step>
</Steps>

## Whitelisting a User

<Info>
  You must have a product
</Info>

<Steps>
  <Step title="Run the Command" icon="command" stepNumber={1}>
    ```
    /add-whitelist <roblox-user-id> <product>
    ```

    If you dont know the user ID run /getinfo and enter there username to get there information - such as there User ID.

    <Warning>
      The product name should be **exatcly** as it was name, **case sensative**
    </Warning>
  </Step>
